## 2. Pairs

**PAIR-A — Long INTC (CND-001) / Short ARM (CND-007)**

Pair type: relative value within GICS sub-industry 45301020 (Semiconductors). The pair encodes the x86 server CPU socket-share recovery thesis versus ARM-architecture server royalty growth — both legs in the same GICS sub-industry, keeping the trade sector-neutral at the semiconductor level.

What the pair isolates: if x86 (Intel Xeon 6) recaptures AI inference sockets from ARM-based hyperscaler custom silicon (Microsoft Cobalt, Google Axion, AWS Graviton), INTC benefits directly while ARM's server royalty growth rate slows relative to its current trajectory. The TAM expansion itself (WC-005) is net positive for both, so the pair must be sized notionally equal to isolate the relative share-shift mechanism, not the absolute TAM lift.

Sizing note: ARM's total royalty revenue ($939M FY2025) includes substantial non-server segments (embedded, mobile). A mismatch in notional size against a large INTC long would leave residual semiconductor-cycle beta in the trade. Equal-notional sizing is the research swarm's task to confirm.

Asymmetry risk: if the AI TAM expands broadly without x86 recapturing share from ARM, both legs move in the same direction. This pair is not beta-neutral against a broad AI-infrastructure rally — it is architecture-share-neutral only.
