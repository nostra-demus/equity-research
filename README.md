# Equity Research

A personal repository for tracking equity research, investment theses, and the mental models I use to evaluate companies.

## Organization

- **`/analyses/`** — Individual ticker analysis reports written in markdown. One file per company, deep dives into business model, financials, competitive position, and valuation.
- **`/watchlist/`** — Tickers I'm actively monitoring but haven't taken a position in. Notes on what I'm waiting for (price, catalyst, thesis confirmation).
- **`/frameworks/`** — My investing checklists, mental models, and repeatable processes. The lens I bring to any new opportunity.
- **`/positions/`** — Current holdings with thesis tracking. Why I own it, what would change my mind, and how the thesis is playing out over time.

## Conventions

- One markdown file per ticker, named with the symbol (e.g., `AAPL.md`).
- Date entries when updating a thesis or analysis so the evolution is visible in git history.
- Keep frameworks generic and reusable; keep analyses specific and evidence-based.
